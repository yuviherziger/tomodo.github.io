# tomodo tags

Manage image tags.

```shell
tomodo tags --help
```

## Subcommands

***

### tomodo tags list

List image tags

#### Usage

```shell
tomodo tags list
```

#### Options and Flags

**`--version`**

- **Description:** The deployment's start port
- **Type:** Filter by version. Must be a valid semantic version (whole or partial).
- **Required:** No
- **Example:**

    ```shell
    tomodo tags list --version 7.0.6
    ```
